# Story 3.1: Build "Metro Map" Component

**Epic:** [Epic 3: Dashboard & Gamification](../epics/epic-3-dashboard-gamification.md)
**Priority:** High
**Status:** Open

## Description
Create the visual "Metro Map" component that serves as the main dashboard navigation.

## Acceptance Criteria
- [ ] SVG or HTML/CSS representation of a 10-stop line.
- [ ] Node States:
    - **Locked:** Gray, non-clickable.
    - **Active:** Pulsing/Highlighted, clickable (leads to Chat).
    - **Completed:** Green checkmark/Color, clickable.
- [ ] Responsive design (vertical on mobile, horizontal on desktop).

## Technical Notes
- Consider `framer-motion` for smooth transitions.
